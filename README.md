# SearchEngine-with-groq-and-tavily
AI-Powered Search & Summarization Engine:
This project is an AI-driven search engine with a modern UI that delivers accurate, concise, and easy-to-understand answers, similar to Perplexity. It combines real-time web search with intelligent summarization to help users quickly grasp the most relevant information without information overload.
By integrating Tavily for web search and Groq for high-speed LLM inference, the system retrieves reliable sources, processes them intelligently, and presents clear, summarized responses in a user-friendly interface.

 Key Features:
 Perplexity-Style AI Search: Retrieves information from the web and answers queries contextually
 Smart Summarization: Condenses multiple sources into clear, concise explanations
 Powered by Tavily + Groq: Real-time search with ultra-fast AI reasoning
 Simple & Precise Explanations: Focused on clarity and easy understanding
 Interactive UI Interface: Clean and intuitive design for smooth user experience

 Use Cases:
Quick research and knowledge discovery
Students and learners who prefer summarized explanations
Professionals needing fast, reliable insights
Anyone who wants search results explained, not just listed
